# AzhuTech

Practical open-source tools for AI-native developers.

AzhuTech builds small, sharp developer tools for AI-assisted engineering, automation, and local-first workflows. The first focus is helping teams make coding agents safer, more consistent, and easier to adopt in real repositories.

## Projects

- `agent-rules`: generate repository-specific instructions for AI coding agents.

## Principles

- Useful in five minutes.
- Clear enough to understand from one README.
- Built for real workflows, not demos alone.
- Maintained with fast feedback loops and contributor-friendly issues.
